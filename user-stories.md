# FitTogether – User Stories

## Crowdsourced Busyness

1. ✅ As a **user**, I can see a list of local pools/gyms with their current busyness level (Low/Medium/High) so I know when to go.
2. ✅ As a **user**, I can check in at a location and report how busy it is right now.
3. ✅ As a **user**, I can see when the last busyness report was submitted (timestamp) to judge how fresh it is.
4. ✅ As a **user**, I can view locations on an interactive map with color-coded markers (green/yellow/red by busyness).
5. ✅ As a **user**, I can search for places by name using Nominatim (OpenStreetMap) and add them as locations.
6. ✅ As a **user**, I can see visual crowding indicators — color-coded borders, emoji icons, capacity bars, and warning banners for overcrowded venues.
7. ✅ As a **user**, I can click a location to view its detail page with focused map, current status, and recent check-in reports.

## Buddy Finder

8. ✅ As a **user**, I can set my preferred activities (swim/gym) and workout times (morning/evening/weekend) on my profile.
9. ✅ As a **user**, I can browse other users and filter by activity or workout time to find compatible partners.
10. ✅ As a **user**, I can write a short bio so others know a bit about me.
11. ✅ As a **user**, I can send a buddy request to another user to connect.
12. ✅ As a **user**, I can accept or decline incoming buddy requests from my profile page.
13. ✅ As a **user**, I can see the status of my buddy connections (pending, accepted) on the buddies page.
14. ✅ As a **user**, I can contact an accepted buddy via WhatsApp — but only when they have set their status to "at the gym" or "at the pool".
15. ✅ As a **user**, I can set my active status (🏋️ At the Gym / 🏊 At the Pool / ⚫ Offline) on my profile to control when buddies can contact me.
16. ✅ As a **user**, I can see a green dot indicator on buddies who are currently active at a venue.

## Challenges

17. ✅ As a **user**, I can browse available challenges and see how many people have joined.
18. ✅ As a **user**, I can join a challenge (e.g., "Swim 5km in a week").
19. ✅ As a **user**, I can log workouts to track my progress toward a challenge goal.
20. ✅ As a **user**, I can view a challenge's leaderboard to see who's leading.
21. ✅ As a **user**, I can see my progress bar and percentage toward completion.

## Profile & Auth

22. ✅ As a **user**, I can sign in with Google (one click) via Auth.js.
23. ✅ As a **user**, I can view my profile with check-in history and joined challenges.
24. ✅ As a **user**, I can edit my activities, workout times, WhatsApp number, and bio.
25. ✅ As a **user**, I can see and respond to incoming buddy requests on my profile page.

## Safety & Anti-Harassment

26. ✅ As a **user**, my WhatsApp number is hidden until I both accept a buddy request AND set myself as active — preventing unwanted contact.
27. ✅ As a **user**, I can reject buddy requests from people I don't want to connect with.
28. ✅ As a **user**, I can go offline at any time to stop all incoming WhatsApp contact from buddies.

## Future

29. As an **admin**, I can create new challenges from a dashboard.
30. As a **user**, I get notified when a buddy request is accepted.
31. As a **user**, I can see historical busyness trends (e.g., "usually busy at 5pm on Mondays").
32. As a **user**, I can earn badges for completing challenges.
33. As a **user**, I can block another user entirely.
