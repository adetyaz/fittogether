# FitTogether – User Stories

## Crowdsourced Busyness

1. As a **visitor**, I can see a list of local pools/gyms with their current busyness level (Low/Medium/High) so I know when to go.
2. As a **signed-in user**, I can check in at a location and report how busy it is right now.
3. As a **visitor**, I can see when the last busyness report was submitted (timestamp) to judge how fresh it is.
4. _(Future)_ As a **visitor**, I can view locations on a map (Leaflet/Google Maps).

## Buddy Finder

5. As a **signed-in user**, I can set my preferred activities (swim/gym) and workout times (morning/evening/weekend) on my profile.
6. As a **signed-in user**, I can browse other users and filter by activity or workout time to find compatible partners.
7. As a **signed-in user**, I can "Say Hi" to another user, which copies their email so I can reach out.
8. As a **signed-in user**, I can write a short bio so others know a bit about me.

## Challenges

9. As a **visitor**, I can browse available challenges and see how many people have joined.
10. As a **signed-in user**, I can join a challenge (e.g., "Swim 5km in a week").
11. As a **signed-in user**, I can log workouts to track my progress toward a challenge goal.
12. As a **visitor**, I can view a challenge's leaderboard to see who's leading.
13. As a **signed-in user**, I can see my progress bar and percentage toward completion.

## Profile & Auth

14. As a **visitor**, I can sign in with Google (one click).
15. As a **signed-in user**, I can view my profile with check-in history and joined challenges.
16. As a **signed-in user**, I can edit my activities, workout times, and bio.

## Future

17. As a **user**, I can send/accept buddy requests instead of just copying emails.
18. As an **admin**, I can create new challenges from a dashboard.
19. As a **user**, I get notified when a buddy request is accepted.
20. As a **user**, I can see historical busyness trends (e.g., "usually busy at 5pm on Mondays").
21. As a **user**, I can earn badges for completing challenges.
